# Настройка Firebase

Пошаговая инструкция по настройке бэкенда.

### 1. Создание проекта
1.  Перейдите в [Firebase Console](https://console.firebase.google.com/).
2.  Нажмите **"Add project"** и следуйте инструкциям.
3.  Включите **Google Analytics** (опционально).

### 2. Аутентификация
1.  В меню слева выберите **Build > Authentication**.
2.  Нажмите **"Get started"**.
3.  Вкладка **"Sign-in method"** > Нажмите **"Email/Password"** > Включите и сохраните.

### 3. Firestore Database
1.  В меню выберите **Build > Firestore Database**.
2.  Нажмите **"Create database"**.
3.  Выберите местоположение (ближайшее к РФ — `europe-west`).
4.  Начните в **"Production mode"**.
5.  После создания перейдите во вкладку **Rules** и скопируйте содержимое файла `firestore.rules` из корня вашего проекта.

### 4. Назначение Администратора
1.  Зарегистрируйтесь на сайте как обычный пользователь.
2.  Перейдите в Firebase Console > Firestore Database > Коллекция `users`.
3.  Найдите документ с вашим UID (можно посмотреть в Auth).
4.  Добавьте поле: `isAdmin` (boolean) со значением `true`.
5.  Теперь при входе на сайт в меню профиля появится ссылка на **Админ-панель**.

### 5. Регистрация Веб-приложения
1.  Project Overview > Нажмите на иконку `</>` (Web).
2.  Введите название.
3.  Скопируйте объект `firebaseConfig` и вставьте значения в ваш `.env.local`.
