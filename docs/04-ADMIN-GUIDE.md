# Руководство Администратора

Инструкция по управлению магазином через админ-панель.

### 1. Доступ
Доступ к админке осуществляется по адресу `/admin`. 
**Важно**: Вы должны быть авторизованы и иметь поле `isAdmin: true` в вашем документе в коллекции `users`.

### 2. Разделы
*   **Дашборд**: Общая статистика выручки, активных заказов и клиентов.
*   **Заказы**: Список всех заказов. Здесь можно менять статусы.
    *   `Новый` -> `В работе` -> `Отправлен` -> `Доставлен`.
    *   При переходе в статус `Доставлен` система автоматически пересчитывает VIP-уровень клиента на основе его общей суммы покупок.
*   **Товары**: Управление каталогом (добавление, удаление, редактирование цен и скидок).
*   **Финансы**: Модуль ручного учета. Записывайте закупки и продажи, не связанные с заказами напрямую. Есть экспорт в XLSX для отчетности.
*   **Отзывы**: Модерация отзывов. Отзыв появится на сайте только после вашего одобрения (`Approve`).
*   **Настройки**: Управление порогами VIP-скидок и полями формы заказа.

### 3. Инициализация (Seed)
Если вы только запустили проект и база пуста, перейдите в **Настройки** и нажмите **«Инициализировать БД (Seed)»**. Это создаст базовые товары и конфигурацию.
