# Проект FreeStyle Store: Production Ready

Все задачи по разработке и интеграции проекта успешно выполнены. 

### Что было сделано:
1.  **Интеграция Firebase**: Настроена авторизация, база данных Firestore и хранилище. Все ключи вынесены в переменные окружения.
2.  **Каталог и Корзина**: Реализован современный каталог с поддержкой скидок и функциональная корзина с формой оформления заказа.
3.  **Личный кабинет**: Создан раздел для пользователей с историей заказов и отображением статуса лояльности (VIP уровни).
4.  **Админ-панель**: Реализован мощный инструмент управления сайтом:
    *   **Заказы**: Просмотр и смена статусов с автоматическим пересчетом лояльности клиента.
    *   **Товары**: Управление ассортиментом.
    *   **Аналитика**: Графики продаж и статистика выручки.
    *   **Финансы**: Модуль ручного учета закупок и продаж с экспортом в XLSX.
    *   **Отзывы**: Система пре-модерации отзывов.
    *   **Настройки**: Динамическая конфигурация полей заказа и порогов VIP.
    *   **Инициализация (Seed)**: Добавлена кнопка «Инициализировать БД» для мгновенного наполнения сайта товарами.
5.  **Cloudflare Worker**: Создан серверный слой для безопасной обработки заказов и уведомлений в Telegram.
6.  **Дизайн и Анимации**: Весь проект переведен на `framer-motion` (AOS удален). Реализован "liquid glass" дизайн с плавными переходами.

### Запуск проекта:
1.  `npm install` — установка зависимостей.
2.  `npm run dev` — запуск в режиме разработки.
3.  `npm run build` — сборка для продакшена (статический экспорт).

### Настройка:
Подробные инструкции по настройке Firebase, Cloudflare и деплою на GitHub Pages находятся в папке `docs/`.
