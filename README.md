# FreeStyle Libre Shop

Современный интернет-магазин сенсоров FreeStyle Libre с личным кабинетом и админ-панелью.

## Возможности
- **Лендинг**: Hero-секция, преимущества, каталог, сравнение Libre 2 и 3, инструкция по установке, FAQ.
- **Корзина**: Динамический расчет стоимости с учетом скидок лояльности.
- **Личный кабинет**: История заказов, профиль, уровни VIP.
- **Админ-панель**: Управление заказами (статусы), товарами, пользователями, модерация отзывов, настройки лояльности и Telegram-бота.
- **Уведомления**: Мгновенные оповещения админа в Telegram при новом заказе.
- **Дизайн**: Темная/светлая темы, анимации AOS, адаптивность.

## Технологии
- Next.js 15 (App Router)
- Firebase (Auth, Firestore, Functions)
- Tailwind CSS + Shadcn UI
- Chart.js (Аналитика)

## Настройка Firebase
1. Создайте проект в [Firebase Console](https://console.firebase.google.com/).
2. Включите **Authentication** (Email/Password).
3. Включите **Cloud Firestore**.
4. Включите **Firebase Functions** (требуется план Blaze для внешних запросов к Telegram).
5. Создайте коллекцию `settings` с документом `config`.
6. Укажите API ключи в `.env.local` (см. список ниже).

## Команды
- `npm install` - установка зависимостей.
- `npm run dev` - запуск в режиме разработки.
- `npm run build` - сборка проекта.
- `firebase deploy --only functions` - деплой облачных функций.
